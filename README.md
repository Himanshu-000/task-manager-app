# Task Manager App 📝

A modern Task Manager web application built using HTML, CSS, and JavaScript.

## 🚀 Features
- Add, Edit, Delete Tasks
- Mark tasks as completed
- Filter tasks (All / Completed / Pending)
- Local storage support

## 🛠 Tech Stack
- HTML
- CSS
- JavaScript
